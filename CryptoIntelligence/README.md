# CryptoIntelligence/

Crypto-specific analysis, risk detection, and defensive intelligence — part of the BARREN WUFFET system.

## Key Modules

| Module | Purpose |
|--------|---------|
| `crypto_intelligence_engine.py` | Core engine: venue health, counterparty risk, route optimization |
| `scam_detection.py` | Rug pull, honeypot, pump-and-dump pattern detection |
| `mev_protection_system.py` | MEV attack detection and transaction protection |
| `whale_tracking_system.py` | Large wallet monitoring and smart money flow |
| `defi_yield_analyzer.py` | DeFi yield farming analysis, impermanent loss |
| `onchain_analysis_engine.py` | MVRV, SOPR, exchange flows, supply dynamics |
| `crypto_technical_patterns.py` | Funding rates, liquidation cascades, order flow |
