# Strategy Cells & Modularity (v1.0)

## Fractal Org Pattern
AAC strategies are organized as "cells":
- Local autonomy
- Strict boundaries
- Standard reporting
- Global coordination via portfolio layer

## Cell Types
- Trend cell
- Mean-reversion cell
- Volatility cell
- Event-driven cell
- Options-structure cell (if applicable)

## Cell Contract (Must Produce)
- Signal Card
- Regime Card
- Backtest Packet
- Risk Packet
- Weekly performance summary
- Failure analysis when underperforming

## Portfolio Layer
Portfolio coordinates:
- Total exposure
- Correlation cluster risk
- Regime-wide throttles
- Capital reallocation
