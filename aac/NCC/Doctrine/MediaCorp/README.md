## 🚀 Quick Start

### Create Your First Avatar
```powershell
# Run the avatar creation script
.\MMC.AvatarCreation.ps1 -VideoPath "C:\path\to\your\video.mp4" -VoicePreset "Adam" -DeploymentTarget "Web"
```

### Available Parameters
- `-VideoPath`: Path to your 2-minute source video
- `-VoicePreset`: "Adam" or "Antoni" (default: Adam)
- `-AutonomousMode`: Enable AI-driven autonomous operation
- `-DeploymentTarget`: "Web", "ZOOM", or "OBS" (default: Web)
