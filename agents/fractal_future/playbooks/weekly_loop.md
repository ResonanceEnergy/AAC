# Weekly Loop Playbook — FRACTAL FUTURE Swarm

## Trigger
Every week (suggest Sunday evening or Monday morning).

## Steps

### 1. Entropy Review (10 min)
- Sentinel produces weekly entropy summary across all three domains
- Governance cell reviews — any valve trips this week?
- Set entropy posture for coming week

### 2. Synthesis Pass (30 min)
- Synthesis cell processes research backlog
- Write/update FF-NOTEs (assign evidence grades E0–E2)
- Update pattern taxonomy if new types discovered
- Write to NCL memory layer with decay tier

### 3. Governance Audit (20 min)
- Review all pending claims
- Run gate checks (FF-1 through FF-6) on any promotion requests
- Mark falsified/weakened claims
- Log all decisions

### 4. Regime Map (AAC Integration) (15 min)
- Fill weekly_regime_map.md template
- Primary regime + entropy regime + feature votes
- Strategy cell posture recommendations

### 5. Publishing Queue Review (10 min)
- Any material ready for external publishing? (E2+ only)
- Prioritize publishing cell backlog

### 6. Scorecard (5 min)
- Fill FF_SCORECARD template
- Pattern count, evidence upgrades, experiments active, entropy state

## Abort Conditions
- If entropy was E4+ for >3 days this week: abbreviated loop (governance audit only)
- Sentinel failure during loop: pause and reschedule
