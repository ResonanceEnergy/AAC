# Monthly Loop Playbook — FRACTAL FUTURE Swarm

## Trigger
First week of each month.

## Steps

### 1. Doctrine Delta (30 min)
- Fill monthly_doctrine_delta.md template
- What changed this month?
- What was falsified or weakened?
- What strengthened with evidence?
- Proposed doctrine page updates

### 2. Experiment Review (20 min)
- Experiment cell reports status on all active experiments
- Any experiments to close, extend, or start?
- Evidence grade updates from experiment results

### 3. Full Entropy Audit (15 min)
- Sentinel produces monthly entropy trend analysis
- Entropy budget spend vs. allocation
- Valve activation history + effectiveness review
- Entropy map refresh (both markets + attention)

### 4. NCL Memory Prune (15 min)
- Review NCL memory layer
- Prune L3 (stale) items older than 90 days with no refresh
- Promote items with supporting evidence
- Demote items contradicted by new data

### 5. NCC Gate Batch (20 min)
- Governance cell runs batch gate review on all pending promotions
- Feed results back to NCC Doctrine registry

### 6. Publishing Sprint (variable)
- Publishing cell processes approved queue
- Produce monthly publishing brief

### 7. Roadmap Check (10 min)
- Are we on track for current phase goals?
- Adjust priorities for next month
- Update prioritization matrix if needed

## Abort Conditions
- If system entropy was E4+ for >1 week: reschedule to next week
- Monthly loop is mandatory — cannot be skipped, only deferred
